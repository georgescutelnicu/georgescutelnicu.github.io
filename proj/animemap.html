AnimeMap-Screen2Scene

AnimeMap-Screen2Scene is a web application that maps real-world locations featured in anime. Users can explore a map with anime-related locations and view details about the anime and their corresponding scenes.

► The project can be explored here.

► How did i scrape the data can be seen here.

► GitHub repository can be seen here.

The project is hosted on Render.
Please note that it may take up to 50 seconds to start when first accessed, as I am using a free instance on Render.

Special thanks to フラバーのふらふらアニメ聖地巡礼 for their amazing work in mapping anime pilgrimage locations.

Technologies used

  • Backend: Flask, SQLAlchemy
  • Frontend: HTML, CSS, JavaScript, Leaflet.js
  • Database: PostgreSQL (via SQLAlchemy ORM)
  • Hosting: Render
